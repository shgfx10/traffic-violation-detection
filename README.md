# 🚦 Traffic Violation Detection System

An AI-powered real-time traffic violation detection dashboard built with HTML, CSS, and JavaScript.

## Features
- 🎥 Live camera feed simulation (6 cameras)
- 🚗 Real-time vehicle tracking with bounding boxes
- ⚡ AI violation detection (Overspeeding, Signal Jump, Lane Violation, Wrong Side, No Helmet, Illegal Parking)
- 📊 Live analytics charts (violation types & hourly trends)
- 🧾 Challan/Fine issuing system
- 📍 Location-based camera monitoring

## Tech Stack
- HTML5 Canvas
- CSS3 Animations
- Vanilla JavaScript
- Real-time simulation engine

## How to Run
Just open `traffic_violation_detection_system.html` in any browser!
